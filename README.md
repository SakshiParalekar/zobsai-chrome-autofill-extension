# Job Application Autofill Chrome Extension

## Overview
A simple Chrome extension that autofills common job application fields using dummy data.
Built as a technical assignment for Zobsai.

## Features
- Autofills name, email, phone number, and skills
- One-click autofill using popup button
- Uses Chrome Extension Manifest V3

## Setup & installation steps
1. Open Chrome and go to chrome://extensions
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select the project folder

## How to Use the extension
1. Open any sample job application form
2. Click the extension icon
3. Click "Autofill Form"
